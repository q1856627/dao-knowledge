# 三元八会文

> 三元八会之文，又名"道祕天书"、"无字天书"，是上古道学传承中的古文字体系。

## 什么是三元八会文

三元八会文从**天地人三才**与**八会之理**解读汉字，是字典与思想文化典籍的合一。

- **三元**：天元、地元、人元
- **八会**：八方会聚之理

- **宇宙大人身** — 宇观层面的整体观
- **人体小宇宙** — 微观层面的全息观
- **宇宙的全息性** — 宏微观的同构关系
- **雷形电气** — 自然现象与道学解读
